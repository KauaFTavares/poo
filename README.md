## Projeto de Orientação a Objetos

Esse projeto gerencia informações dos empregados, permitindo adcionar, buscar, listar e armazenar informações dos empregados; esses dados são desde: nome, email, telefone,alergias e problemas medicos que permitem um entendimento de cada membro da empresa.

## Estrutura

O projeto conta com um esqueleto de
src
* Modelo
    Empregado.java
* Servico
    GerenciadorDeEmpregados.java
* Usuari
    PainelUsuario.java
* Condicoes.java

## Configuração do Ambiente

1 Instale o Java Development Kit (JDK)
2 Instale como editor de código o VsCode
